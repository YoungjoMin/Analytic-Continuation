Detailed Description

complex function of power series with domain

Remarks
POWERSERIES is a class of power series function defined on complex plane.

POWERSERIES has three member variable coef, centor, k.
coef is a function which gets nonnegative integer n and returns n-th coefficient
COMPLEX z0 is centor of the power series .
k is positive integer about size of coefficient and the domain of the power series. it should be given when the POWERSERIES is initialized.
with given k, coefficient should satisfy

then POWERSERIES is function defined on a circle.
